# Online Complaint Registration System

## Project Overview

The Online Complaint Registration System is a MERN Stack web application that allows users to register complaints, track complaint status, and provide feedback after resolution. The system improves transparency and simplifies complaint management.

## Features

* User Registration and Login
* JWT Authentication
* Complaint Submission
* Complaint Tracking
* Status Management
* Admin Dashboard
* Feedback System
* MongoDB Database Integration

## Technology Stack

### Frontend

* React.js
* Bootstrap
* Axios

### Backend

* Node.js
* Express.js

### Database

* MongoDB
* Mongoose

## Installation

### Backend

```bash
npm install
npm start
```

### Frontend

```bash
npm install
npm run dev
```
